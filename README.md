# Smart HOME 프로젝트
메타빌드 제출용 프로젝트

## Description
Smart HOME을 주제로 IoT 센서로 부터 실시간 데이터를 전송받아 대쉬보드를 꾸민다

## Functions
- Admin (User MGMT, Settings)
- Realtime Dashboard
- Resource Monitoring
- Alarm Settings
- Account

## Install & Test
For frontend
```
npm install
npm run start
```

For backend
```
mvn clean package
```
